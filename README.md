# TimerTodo

iOS-приложение для управления задачами и концентрации с использованием таймера Pomodoro. Помогает фокусироваться на задачах, эффективно распределять время и отслеживать выполненные дела. Минималистичный интерфейс и упор на продуктивность.

## Основные требования и особенности проекта

**Архитектура:** SwiftUI + MVVM с чистым разделением слоёв 
**Хранение данных:** локально (TaskService)  
**Логика фокуса:** Pomodoro-таймер и состояния фокусировки  
**Утилиты:** Logger для отладки и отслеживания событий 

**Основные экраны приложения:**
1. Home - основной экран с таймером и быстрым добавлением задач
2. Tasks — список задач (добавление, выполнение)
2. Focus — Pomodoro-таймер для фокусированной работы, игра в перерыве
4. Quotes — мотивационные цитаты
5. FAQ — раздел инфо о приложении

## Описание приложения (для App Store Connect)

**Название:** TimerTodo  
**Подзаголовок:** Фокус, задачи и Pomodoro-таймер  

**Описание:**  
TimerTodo — это простое и эффективное iOS-приложение для управления задачами и повышения концентрации. Используйте Pomodoro-таймер, чтобы работать без отвлечений, отмечайте выполненные задачи и поддерживайте мотивацию с помощью вдохновляющих цитат.

**Особенности:**
- Список задач с локальным хранением
- Pomodoro-таймер для фокусированной работы
- Отслеживание состояния фокуса
- Мотивационные цитаты
- Быстрый и отзывчивый интерфейс
- Светлая и тёмная тема
- Полностью адаптивный интерфейс для iPhone

**Ключевые слова:** todo, pomodoro timer, focus, productivity, task manager  

**Возрастной рейтинг:** 4+  
**Категория:** Продуктивность
